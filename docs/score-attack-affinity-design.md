# スコアアタック向け相性設計メモ

この文書は、将来的にスコアアタック形式の対人戦を導入する場合に、`出来事`、`役者型`、`役者状態`、`公演回` の相性表示へどのようなゲーム的深みを持たせるかを整理した設計メモです。

現時点では実装対象ではありません。画面、ルール、対人戦形式、公演スコアの方向性がもう少し固まった後で参照するための将来案です。

## 背景

現在の行動に対応フェーズでは、初心者は `見立て` と `影響` を見れば大きく外さずに遊べる。一方で、相性表示は主に `見立て` の内訳説明になっており、上級者が相性を読んで戦略を分岐させる余地はまだ小さい。

スコアアタック勝負を想定する場合、単に計算を複雑にするだけではなく、プレイヤーが「次は違う読み筋を試したい」と思える複雑性が必要になる。相性4軸を単なる加点ではなく、それぞれ異なる戦略資源へ接続すると、繰り返し遊ぶ価値が生まれやすい。

## 基本方針

- 初心者は `見立て` と `影響` だけで遊べる。
- 中級者は相性を見て、どの対応が何に強いかを読む。
- 上級者は相性ごとの蓄積、倍率、リスク管理を使って同じ seed のスコアを詰める。
- 相性は総合点へ吸収されるだけでなく、結果の質、次ターン以降の伸び方、終盤倍率、事故リスクへ意味を持つ。

## 相性4軸の将来役割

### 出来事相性

今ターンの瞬間火力を担う軸。

- 今起きた本番の出来事に、選んだ対応がどれだけ効くかを見る。
- `◎` はその場の評判スコア、名場面到達、上振れ幅に強く効く。
- スコアアタックでは「このターンで点を取りにいくか」を判断する主軸になる。

読み筋の例:

- `出来事◎` は今の揺れを見せ場に変える攻め筋。
- `出来事×` は今の出来事と逆方向の手で、点を取りにくい。

### 役者型相性

役者らしさと信頼成長を担う軸。

- 主役、若手、技巧派という役者型に対応が合っているかを見る。
- 将来的には個別信頼、座組信頼、役者別ハイライト、終盤ボーナスへ接続する。
- `◎` や `○` は、即時点だけでなく「この役者を活かした公演」へ育てる意味を持つ。

読み筋の例:

- 若手に `拾う` が合うなら、勢いを評判へつなげる。
- 主役に `待つ` が合うなら、終盤で信頼や余韻の伸びを狙う。
- 技巧派に `整える` が合うなら、公演全体の精度を高める。

### 役者状態相性

安全性、負荷制御、下振れ抑制を担う軸。

- 高揚、沈思、不安、没入、疲労といった現在状態に対応が合っているかを見る。
- 将来的には負荷増減、事故率、下振れ幅、疲労、ほころび発生率へ接続する。
- `◎` や `○` は、同じ見立てでも安全に通せる手として意味を持つ。

読み筋の例:

- 疲労時に `整える` や `切る` が合うなら、事故化を防ぐ。
- 不安時に `整える` が合うなら、負荷やほころびを抑える。
- 高揚時に無理に `待つ` と、勢いを殺して下振れやすい。

### 公演回相性

長期プラン、公演の型、日程倍率を担う軸。

- 1日目、2日目、3日目、マチネ、ソワレというタイミングに対応が合っているかを見る。
- 将来的には公演の型、日跨ぎ回復、ソワレ倍率、千秋楽倍率、次ターンの安定度へ接続する。
- `◎` や `○` は「今このタイミングで選ぶ意味がある」ことを示す。

読み筋の例:

- 初日は `待つ` や `整える` で呼吸と型を作る。
- 2日目は揺らぎを攻めるか、負荷を整えるかの分岐を作る。
- 千秋楽は残った信頼や負荷管理をスコアへ変換する。

## 見立てとの関係

将来的には以下の役割分担を目指す。

- `見立て`: 今ターンの結果幅を示す初心者向けの総合目安。
- `相性`: その結果がどんな質を持つか、どの将来資源へつながるかを示す。
- `影響`: 選択後に負荷、信頼、流れへ何が起きるかを示す。
- `準備`: 事前に選んだ準備が本番中の対応をどれだけ支えているかを示す。

つまり、見立てが同じでも相性の内訳によって意味が変わるようにする。

例:

- `出来事◎ / 役者状態×`: 爆発力はあるが危ない攻め手。
- `出来事○ / 役者状態○ / 公演回○`: 上限は控えめだが安定して次へ渡せる手。
- `役者型◎ / 公演回○`: 今ターンだけでなく終盤ボーナスを仕込む手。

### 現行の見立てレンジ

現行実装の `見立て` は、ランダム抽選の確率幅ではなく、評価上の安全域と上振れ余地をまとめたレンジである。

- 相性、準備、信頼、負荷、連続使用などから基準スコアを作る。
- 基準スコアから中心結果を決める。
- 準備が活きた場合は上振れ幅が広く、名場面まで届きやすい。
- 準備が一部活きた場合や別の備えだった場合は上限が抑えられ、攻め手でも伸び切らない。
- 一部の守り手は、準備と噛み合うと事故側の下振れを抑える。

UI上は `名場面狙い / 小成功〜名場面` のように、狙いとレンジを一体で見せる。カード上では比較に必要な最小情報として `準備活きる`、`備え外`、`準備合わず` のような短い言葉を表示し、詳細な意味は進行卓HUDや結果画面で説明する。

### 将来的な改善余地

スコアアタックや対人戦で深みを出す場合、見立てレンジを次のように分解できる。現行方針ではランク判定に確率を入れないため、期待値ではなく判断の性質として表示する。

- `伸びしろ`: 名場面や高得点に届く余地。
- `安定`: 下振れや事故をどれだけ抑えられるか。
- `代償`: 負荷や段取り、一体感に何を残すか。
- `再現性`: 同じ読み筋を繰り返した時にどれだけ安定して通るか。

この段階ではカード表示を増やしすぎず、初心者向けには `見立て`、上級者向けにはHUDや詳細画面で `伸びしろ / 安定 / 代償` を読める構成が望ましい。

## スコアアタックで期待する戦略型

### 瞬間火力型

出来事相性を重視して、そのターンの場面点と名場面到達を狙う。

### 信頼育成型

役者型相性を重視して、座組信頼や役者別ボーナスを育て、終盤に伸ばす。

### 安定運用型

役者状態相性を重視して、負荷、事故、ほころびを抑え、平均点を高く保つ。

### 公演設計型

公演回相性を重視して、初日で型を作り、2日目で分岐し、千秋楽で倍率を乗せる。

## 対人スコアアタックとの相性

同じ seed を使って複数プレイヤーが同じ公演を回す形式では、この設計が活きやすい。

- 同じ出来事に対して、攻める人と整える人で分岐する。
- 短期点を取った人が後半の負荷で苦しむ。
- 信頼を育てた人が終盤に伸びる。
- 初日ソワレの型選択で以降の戦略が変わる。
- 同じ seed の再演で最適化が進む。

## UIへの含意

現時点のUIでは、相性は `見立て` の説明材料に見えやすい。将来、相性に戦略資源としての意味を持たせるなら、UIでも以下を表現できるとよい。

- `出来事相性`: このターンの伸び。
- `役者型相性`: 信頼、役者らしさ、終盤ボーナス。
- `役者状態相性`: 安全性、負荷制御、下振れ抑制。
- `公演回相性`: 型、日程、次ターン以降への流れ。

カード上は短く、選択中カードの見立てや結果画面で「どの相性が何に効いたか」を説明するのが望ましい。

## 実装タイミング

この設計は、以下が固まってから実装検討する。

- 対人戦形式がスコアアタックでよいか。
- 最終スコアの評価軸をどうするか。
- 個別役者信頼をUIやスコアへ出すか。
- 公演の型をどこまで戦略資源にするか。
- seed再演やリプレイ最適化をどの程度重視するか。

それまでは、相性表示は `見立て` の内訳説明として扱い、UI上でも過度に主判断へ押し出しすぎない。
